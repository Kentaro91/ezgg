# EZGG

개인 게임 가이드 & 팁 모음 사이트

**https://ezgg.vercel.app**

## Features

- **D2R 액트 진행 가이드** — 노말/나이트메어/헬 전 액트 동선 및 던전 길찾기
- **직업별 팁** — 공통 진행에 직업 전용 팁 오버레이 (현재 악마술사)
- **룬워드 계산기** — 룬워드 검색 및 장비/소켓 필터
- **진행 체크** — 액트 스텝 체크 + 진행률 표시 (localStorage 저장)
- **다크/라이트 모드**
- **모바일 대응** (PWA)

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router v6

## Development

```bash
npm install
npm run dev
```
